==============================================
    KSeF Panel - Instrukcja instalacji
==============================================

KROK 1: Konfiguracja tokena
--------------------------------------------
Otworz plik "config.json" Notatnikiem (kliknij prawym -> Otworz za pomoca -> Notatnik).

Zmien:
  "ksef_token": ""     ->  "ksef_token": "TWOJ_TOKEN_KSEF"
  "nip": ""            ->  "nip": "1234567890"

Zapisz plik (Ctrl+S).

Nie wiesz jak uzyskac token? Instrukcja w aplikacji: Ustawienia -> Jak uzyskac token?


KROK 2: Instalacja
--------------------------------------------
Kliknij prawym przyciskiem na "INSTALUJ.bat"
Wybierz "Uruchom jako administrator"
Poczekaj na komunikat "INSTALACJA ZAKONCZONA!"


KROK 3: Uruchomienie
--------------------------------------------
Na pulpicie pojawi sie ikona "KSeF Panel".
Kliknij dwukrotnie aby uruchomic aplikacje.


ZMIANA TOKENA PO INSTALACJI:
--------------------------------------------
Sposob 1: W aplikacji -> Ustawienia -> Token KSeF -> wklej nowy -> Zapisz
Sposob 2: Otworz C:\KSeFPanel\config.json Notatnikiem, zmien token, zapisz.


PRZENOSZENIE NA INNY KOMPUTER:
--------------------------------------------
Skopiuj caly folder z pendrive na nowy komputer i uruchom INSTALUJ.bat ponownie.


POMOC:
--------------------------------------------
W aplikacji: menu -> "Jak uzyskac token?" - instrukcja krok po kroku.
